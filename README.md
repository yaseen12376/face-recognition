# # Face Recognition Attendance System

## 🎯 Features
- Real-time face recognition attendance tracking
- First Arrival & Latest Visit dual tracking
- Beautiful web dashboard accessible via network
- Professional Excel export with multi-level headers
- Local JSON storage for reliability

## 🌐 Web Dashboard
Access the dashboard at: `http://[YOUR-IP]:5000`

## 🚀 Usage
1. Run face recognition: `python "face recognition.py"`
2. Start web dashboard: `python web_dashboard.py`
3. View attendance from any device on your network

## 📊 Reports
- Real-time web viewing
- Excel export with beautiful formatting
- Statistics and analytics
